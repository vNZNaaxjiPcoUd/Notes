### change background color and max width

@media only screen {
body {
margin: 2em auto;
max-width: 100%;
color: rgb(120, 120, 120);
background-color: rgb(1, 1, 1);
}
}

### bigger font size

h2 {
font-size: 2.2rem;
margin-top: 1.5rem;
}

### change line-height

h1,
h2,
h3 {
letter-spacing: -0.01em;
line-height: 1.5;
font-weight: 600;
margin-bottom: 0;
}
